Surabaya Dev

A community of developers in Surabaya, Indonesia.

Visi & Misi

Surabaya Dev bertujuan menjadi wadah talenta teknologi dengan memanfaatkanpotensinya untuk meningkatkan skill tech dibidang programming maupun digital teknologi agar dapat saling bersinergi berkolaborasi.

Wadah Belajar

Menjadi wadah belajar pegiat IT untuk bisa terus update mengenai tech industry saat ini.

Koneksi

Memperluas relasi melalui komunitas untuk menjalin kebermanfaatan.

Berbagi Ilmu

Komunitas adalah wadah yang tepat untuk bisa saling berbagi ilmu, pengalaman maupun wawasan tentang update teknologi

Kegiatan

Berbagai kegiatan rutin dan non rutin akan diadakan untuk memperluas relasi dan meningkatkan skill. Beberapa kegiatan yang sering kami lakukan meliputi :

Meetup (Seminar / Webinar)

Kegiatan bersama topik teknologi & programming secara online maupun offline

Uppgrade Skill (Workshop)

Pelatihan internal pengurus berupa workshop untuk pengembangan diri seputar teknikal programming

Kolaborasi Komunitas

Sinergi antar komunitas guna untuk mendukung kegiatan komunitas lokal teknologi di Indonesia

Alasan Harus Gabung Surabaya Dev

Suka Belajar Hal Baru

Mengasah Kemampuan
